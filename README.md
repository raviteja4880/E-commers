##  Overview

**TejaCommerce** is a secure and responsive **MERN stack e-commerce platform** designed for a seamless online shopping experience.  
It features **real-time cart updates**, **email notifications**, **live delivery tracking**, and **role-based dashboards** for Admins, Super Admins, Delivery Partners, and Users.  
Built with a **modern UI** powered by **Bootstrap 5** and **Framer Motion** animations for smooth transitions.

---

##  Tech Stack

| Layer | Technologies |
|--------|---------------|
| **Frontend** | React.js, Bootstrap 5, Framer Motion, React Router |
| **Backend** | Node.js, Express.js |
| **Database** | MongoDB Atlas |
| **Authentication** | JWT (JSON Web Token) |
| **Email Integration** | Nodemailer |
| **Data Visualization** | Recharts |
| **Deployment** | Vercel / Render / Netlify + MongoDB Atlas |
